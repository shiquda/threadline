# Threadline

Threadline is a self-hosted Human-Agent Gateway for durable work context, attention management, and decision closure across heterogeneous agent sessions.

The approved MVP specification is in [docs/mvp-spec.md](docs/mvp-spec.md). The separate UI design brief is in [docs/ui-design-brief.md](docs/ui-design-brief.md).

The project is under active MVP development. Production deployment will use Docker with a persistent SQLite data volume.
